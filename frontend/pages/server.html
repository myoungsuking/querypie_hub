서버 정보를 업로드하고 관리하세요

사용 가이드

Create Server API 가이드

API 엔드포인트

POST /api/external/v2/sac/servers

Request Body 필수 필드
  • name: 서버 이름 (필수) - 예: web-server-01
  • host: 서버 호스트 (필수) - 예: 192.168.1.10
  • sshPort: SSH 포트 (필수) - 예: 22
  • osType: 서버 OS 타입 (필수) - ETC, AWS_LINUX, UBUNTU, CENTOS, RHEL 중 하나
  • ftpPort: FTP 포트 (필수, 기본값: 21)
  • telnetPort: Telnet 포트 (필수, 기본값: 23)
  • vncPort: VNC 포트 (필수, 기본값: 5900)
OS Type 설명

ETC: 기타 Linux 배포판

AWS_LINUX: Amazon Linux

UBUNTU: Ubuntu

CENTOS: CentOS

RHEL: Red Hat Enterprise Linux

⚠️ WINDOWS는 지원되지 않습니다.

주의사항
  • osType에 지정된 값(ETC, AWS_LINUX, UBUNTU, CENTOS, RHEL)이 없다면 자동으로 ETC로 등록됩니다.
  • Windows는 지원되지 않으므로 CSV 파일에 WINDOWS를 입력하지 마세요.

팁: CSV 파일에는 name, host, sshPort, osType만 포함하면 됩니다. ftpPort, telnetPort, vncPort는 기본값으로 자동 설정됩니다.
